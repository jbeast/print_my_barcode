    super_dup(new_name = nil) click to toggle source

    An implementation of dup which allows the name to be changed. Dup the original template. If a name is not passed the name will be changed to “label_template_name copy” dup each of the labels and add it to the new label template Saving the dup will create a whole new record with new ids.

    # File app/models/label_template.rb, line 45
def super_dup(new_name = nil)
  duped = self.dup
  duped.name = new_name || "#{name} copy"
  labels.each do |label|
    duped.labels << label.dup
  end
  duped.save
  duped
end
