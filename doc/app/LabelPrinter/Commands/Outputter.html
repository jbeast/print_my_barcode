    module LabelPrinter::Commands::Outputter

    Allows the receiver to specify a list of commands which can then be formed into valid printer output. This list can include standard and derived commands. The standard commands can be signified by their prefix derived commands need to be a method.

    commands() click to toggle source

    Take a list of commands and execute them. Returns an array of objects. In the case of standard commands will just create a new command. With a derived command it will execute the method. A derived command may itself contain a list of commands so the whole thing is flattened to produce a single array.

    # File app/label_printer/label_printer/commands.rb, line 154
def commands
  @commands ||= [].tap do |c|
    commands_list.each do |command|
      if command.instance_of? String
        c << standard_commands[command].new
      else
        _command = self.send(command)
        c << (_command.instance_of?(Hash) ? _command.values : _command)
      end
    end
  end.flatten
end
