    module LabelPrinter::Commands::Formatting

    A formatting command is one which describes the format of something that will be drawn on the label either a barcode or bitmap. id, x origin and y origin are added as attributes. These are required. An id links the formatting and drawing commands. The x origin and y origin signify where the drawing will be placed on the label.
