    class LabelPrinter::PrintJob::Base

    A print job is an ActiveModel object For a print job to be execute it needs to be valid. For a print job to be valid it needs to meet the following criteria:

    • The printer needs to exist.

    • The label template needs to exist.

    • It needs to have some labels.

    If a print job is valid a DataInput object is created.
